# Contact Angle Processing

Contact angle measurements quantify changes in membrane surface hydrophilicity
during aging. Measurements were performed using a goniometer following standard
sessile drop method protocols.

Per-condition processing:
- 6 replicate measurements per cumulative dose interval
- Mean and standard deviation calculated per interval
- Outlier detection via ±2σ criterion

No separate processing code is required — raw measurements are directly usable.
See thesis Section 3.3.3 for measurement protocol details.
